# Wordle

This is a tool designed to help me cheat at Wordle.

## Usage

### Requirements

Required to run this you need `pyyaml` and `tqdm` installed. The easiest way to do
this is through pip.

```bash
pip install -r requirements.txt
```

or

```bash
pip install pyyaml tqdm
```

### Running

```bash
python wordle.py
```
